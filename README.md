# Fire: Interpreter Programming Language

Rust の簡易版みたいなもの。

## Build and Hello World:
Create source file as `a.flame`
```
println("Hello, World!");
```

build and run.
```
$ ./build.sh -J --run test.fire
```

(debug-build)
```
$ ./build.sh -Jd
```
